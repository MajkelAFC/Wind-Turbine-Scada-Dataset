# Wind Energy Analytics Platform

## Problem Domain
Wind energy generation is highly dependent on environmental factors, primarily wind speed and direction. However, physical turbines often deviate from their theoretical power curves due to wear and tear, curtailment, or sensor anomalies. 

## Project Goal
The objective of this project is to build an automated data pipeline that extracts, transforms, and loads (ETL) Wind Turbine SCADA data. We aim to monitor turbine performance, analyze the gap between theoretical and actual power output, and visualize these metrics to detect potential inefficiencies.

## Tech Stack
*   **Orchestration:** Apache Airflow
*   **Data Warehouse:** PostgreSQL
*   **Data Visualization:** Apache Superset
